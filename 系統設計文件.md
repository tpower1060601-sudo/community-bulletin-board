## 四、管理後台（admin）

### 登入
- 預設密碼：`admin`
- 密碼存於 `localStorage` 的 `bbs_settings.adminPassword`

### 側邊欄導覽順序
1. 📢 公告管理（screen1/2 共用 `bbs_announcements`）
2. 📋 公告管理（直立）（screen13 專用 `bbs_announcements_v`）

4. 🖥️ 螢幕排版（拖拉配置 + 產生 .bat）
5. ⚙️ 系統設定（API Keys、GitHub Token、備份還原）

### 公告管理功能
- 類型：一般公告 / 重要提醒 / 活動通知 / 緊急公佈
- 顯示秒數：每則獨立設定（3～120 秒）
- 富文本內容：粗體/斜體/底線/字級/色票/自訂顏色
- 公告圖片：上傳至 GitHub `data/images/`（需 Token）
- 排程：「一直有效（常駐）」或設定起迄日期時間
- 狀態欄：每 60 秒自動刷新（待發布 / 顯示中 / 已結束）

### 螢幕排版（.bat 產生器）
- 螢幕牆：3 × 2（上排 Y:-1080、下排 Y:0，X:0/1920/3840）
- 支援拖拉放置各 screen 到格子上
- 設定：瀏覽器選擇（Edge/Chrome/自訂路徑）、啟動等待秒、視窗間隔秒（預設 60 秒）
- **一般螢幕（高度 ≤ 1080）**：使用 `--kiosk`
- **特殊螢幕（高度 > 1080，如 screen13）**：使用 `--start-fullscreen --app=`

### 視窗管理
- 兩欄卡片格（win-grid，grid-template-columns:1fr 1fr）
- 顯示標題 + 直接連結（點擊開啟對應 screen）
- 無開關（toggle 已移除）

## 七、公告排程設計

### isAnnActive() 邏輯（screen2/screen13）
```javascript
function isAnnActive(a) {
  if (a.alwaysValid !== false && !a.startDate) return true; // 常駐且無日期
  if (a.alwaysValid) return true;                           // 明確常駐
  const now = new Date().getTime();
  const start = a.startDate ? new Date(a.startDate + 'T' + (a.startTime || '00:00')).getTime() : 0;
  const end   = a.endDate   ? new Date(a.endDate   + 'T' + (a.endTime   || '23:59:59')).getTime() : Infinity;
  return now >= start && now <= end;
}
```

### 自動上下架時序
| 觸發時機 | screen1 | screen2 | screen13 |
|---------|---------|---------|---------|
| 每 60 秒定時 | ✅ `filterActive(_rawAnnList)` | ✅ `render()`（ID 未變則略過） | ✅ 同 screen2 |
| 後台儲存公告 | ✅ `api.on` 即時更新 | ✅ | ✅ |
| 頁面重整 | ✅ 每 30 分鐘 meta refresh | ✅ | ✅ |

## 九、BAT 啟動腳本說明

管理後台「螢幕排版」頁面可產生 `start_display.bat`，範例：

```bat
@echo off
timeout /t 15 /nobreak          ← 等待系統啟動

set EDGE="C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe"
set FLAGS=--no-first-run --no-default-browser-check --disable-translate --disable-infobars

:: 上排
start "" %EDGE% %FLAGS% --user-data-dir="C:\EdgeP1" --window-position=0,-1080 --window-size=1920,1080 --kiosk "https://...screen1/"
timeout /t 60 /nobreak          ← 視窗間隔（預設 60 秒）

:: 特殊螢幕（高度>1080）
start "" %EDGE% %FLAGS% --user-data-dir="C:\EdgeP7" --window-position=0,0 --window-size=1920,2160 --start-fullscreen --app="https://...screen13/"
```

**規則**：
- 高度 ≤ 1080：`--kiosk`（真全螢幕，隱藏標題列）
- 高度 > 1080（如 screen13 的 2160）：`--start-fullscreen --app=`（不被實體螢幕截斷）
- 各視窗使用獨立 `--user-data-dir` 避免 profile 衝突

## 十、已知限制與注意事項

| 項目 | 說明 |
|------|------|
| screen5/6 本地媒體 | web 版 `media()` 回傳 `[]`，需用可公開存取的 URL |
| Twelve Data 免費版 | 不含股市指數（^GSPC 等），故改抓龍頭個股 |
| FMP 免費版 | 同樣不含國際指數 |
| screen1 照片 hotlink | 若 chingjia.com 擋 hotlink，需改本地上傳 |
| rss2json 免費版 | 不可加 `count=` 參數（付費功能），否則回傳 error |
